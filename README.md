## About The Project

Types of `ipsums` present as of now:

- Cat Ipsum
- Lorem Ipsum
- Bacon Ipsum
- Hipster Ipsum
- Zombie Ipsum

### Tech stack

This is built in `rust`

<!-- USAGE EXAMPLES -->

## Usage

Install `rust` on your systems. Run `cargo install --path [a path of your choice]`.

Use the cli:

`loremclipsum [-p [--paragraph] int] [-i [--ipsum] string]`

_example_

`loremclipsum -p 5 -i cat`
